# tRPC x Next.js 13 App Router

A demo showcasing the use of tRPC with the Next.js 13 App Router. Credit to [Jack Herrington](https://www.youtube.com/watch?v=qCLV0Iaq9zU) for his video on this.

## Getting Started

Install packages:

```bash
pnpm i
```

Run dev server:

```bash
pnpm dev
```

Enjoy!
